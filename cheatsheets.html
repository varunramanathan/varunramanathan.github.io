Cheatsheets

  • Some bash commands:
    Restart network service-
    systemctl restart network-manager.service
    Download all pdf files (or files of a specific format) from a webpage-
    wget -r -l1 -A.pdf http://www.example.com/page-with-pdfs.htm
    Switchoff touchscreen: First find the device id for your touchscreen using - xinput --list
    Then suppose your device id is n. Then the command is:
    xinput set-prop n "Device Enabled" 0
    To enable it again, just change the 0 to 1. This command can be used for any input device.
